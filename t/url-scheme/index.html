    Starting Client

  • this is meant to simply start the client, The client is expected to initially display the main menu

    • Open Screens

  • this is meant to start the client to initially display the info screen. Once the info screen is left the client shuts down

  • this is meant to start the client to initially display options screen. Once the option screen is left the client shuts down

    • Trigger Synchronizations

  • this is mean to start the client and to trigger a full synchronization. The synchronization is expected to be ended in the sync done screen. After leaving the sync done screen the client shuts down.

  • this is meant to to start the client and to trigger a full synchronization. Once the synchronization is completed the client has to shut down

    • Register to Participant

  • this is meant to startup the client and to trigger a full synchronization. Partiticipant credentials are provided in this call so the client is also registered and in case a participant change is detected the client has to be wiped. After the synchronization is completed the client is expected to shutdown by itself.

    • Forcing Movelet

  • is meant to start the client to execute a specified movelet. After executing the movelet the client shuts down. If the movelet can not be found the client shuts down too.

  • 3211 (Portal Courdavault PRD)

  • 327 (Portal Equipe mobilité PRD)

  • 8 (Portal Cordier PPR)

    • Forcing Category

  • this is meant to start the client to initially display a provided category. In case the category is not found on the device, the client is expected to display the main menu. The client is expected to shut down once the category is left via back

    • Digging deeper in Category

  • this is meant to start the client to initially display a provided category. In case the category is not found the client shuts down. Otherwise the client is expected to shut down once the category is left via back

    • Executing Movelet

  • is meant to start the client to execute a specified movelet. After executing the movelet the client shuts down. If the movelet can not be found the client displays the main menu
